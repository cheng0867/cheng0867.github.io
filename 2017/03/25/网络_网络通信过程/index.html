网络通信过程

发表于 2017-03-25 | 分类于 网络

网络就是为了联通多方并进行通信用的，即把数据从一方传递给另外一方。如果没有网络，程序都是单机的，即不能和其他电脑上的程序进行通信，为了让在不同的电脑上运行的软件，之间能够互相传递数据，就需要借助网络的功能。

集线器

作用：实现多台电脑连接
流程：

  1. 计算机向集线器发送请求的IP地址
  2. 集线器以广播的形式向其它计算机发送数据包
  3. 如果计算机对应IP地址，则将MAC地址返回；如果非对应IP计算机，则放弃数据包
  4. 集线器以广播的形式返回数据包，发送请求机器接收返回MAC地址

缺点：每个数据包的发送都以广播形式，容易堵塞网络

交换机

作用：

  • 转发过滤：当发送数据的目的节点在MAC地址列表中有映射，则转发到目的节点端口而不是所有端口（如果为广播则转发所有端口）
  • 学习功能：以太网交换机了解每一端口相连设备的MAC地址，并将地址同相应的端口映射起来存放在交换机缓存中的MAC地址表中

流程：

  1. 计算机向交换机发送请求的IP地址
  2. 交换机以广播形式向其它计算机发送数据包
  3. 如果计算机对应IP地址，则将MAC地址返回；如果非对应IP计算机，则放弃数据包
  4. 交换机将对应IP的MAC地址缓存起来，并将结果返回给发送计算机

路由器

路由器（Router）又称网关设备（Gateway）是用于连接多个逻辑上分开的网络

作用：

  1. 判断网络地址
  2. 选择IP路径

特点：

  • 不在同一网段的PC，需要设置默认网关才能把数据传送过去
  • 当路由器收到其它网段的数据包时，会根据“路由表”来决定，把该数据包发送到哪个端口；路由表设定有静态和动态方法
  • 每经过一次路由器，那么TTL值就会减一
