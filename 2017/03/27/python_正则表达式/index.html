正则表达式

发表于 2017-03-27 | 分类于 Python

正则表达式（英语：Regular Expression，在代码中常简写为regex、regexp或RE）使用单个字符串来描述、匹配一系列匹配某个句法规则的字符串。

re模块

python中的re模块提供了正则表达式的相关方法

使用方法：

match()

匹配开头的字符串，如果没有找到返回None

使用方法：

group()

提取匹配数据

参数为0：默认值，返回全部匹配结果
参数为n：返回第n个匹配的分组

groups()

返回所有分组，放到一个元组中

search()

从左到右，找到第一个符合正则的内容

使用方法：

findall()

找到所有符合正则的内容

匹配字符

单字符匹配

字符 功能 等价关系
. 匹配任意1个字符（除了\n）
[] 匹配[]中列举的字符
^ 取反匹配[]中列举的字符
\d 匹配数字 [0-9]
\D 匹配非数字 [^0-9]
\s 匹配空白 空格、tab键
\S 匹配非空白
\w 匹配单词字符 [a-zA-Z0-9_]
\W 匹配非单词字符 [^a-zA-Z0-9_]

数量匹配

字符 功能 等价关系
* 匹配字符0次或无限次，即可有可无 {0,}
+ 匹配字符1次或无限次，即至少1次 {1,}
? 匹配字符1次或0次，即1次或没有 {0,1}
{m} 匹配字符出现m次
{m,} 匹配字符至少m次
{m,n} 匹配字符出现m到n次

边界匹配

字符 功能
^ 匹配字符串开头
$ 匹配字符串结尾
\b 匹配一个单词的边界
\B 匹配非单词边界

分组匹配

字符 功能
l 匹配左右任意一个表达式
(ab) 将括号中字符作为一个分组
\num 引用分组num匹配到的字符串
(?P) 分组起名字
(?P=name) 引用别名为name分组匹配到的字符串

贪婪和非贪婪模式

贪婪模式：默认开启，总是匹配尽可能多的字符
非贪婪模式：总是匹配尽可能少的字符

关闭贪婪模式：在需要关闭贪婪模式区域后加?
