字符串 string

定义：双引号或单引号中的数据，就是字符串

字符串基本操作

索引

字符串实际就是字符的数组，所以也支持下标索引

切片

切片是指对操作的对象截取其中一部分的操作

切片语法：[起始:结束:步长]
注：选取的区间属于左闭右开，即从“起始”位开始，到“结束”位前一位结束

反转字符串

字符串常见操作

查找

find()

检测字符是否在字符串中，有返回开始索引值，没有返回-1

mystr.find(str, start=0, end=len(mystr))

*rfind()

类似find(),从右边开始查找

mystr.rfind(str, start=0,end=len(mystr) )

index()

与find()类似，不同：没找到则报错

mystr.index(str, start=0, end=len(mystr))

*rindex()

类似index()，从右边开始

mystr.rindex( str, start=0,end=len(mystr))

count()

查找字符出现的次数

mystr.count(str, start=0, end=len(mystr))

startswith()

检查是否以指定字符开始，返回True/False

mystr.startswith(obj)

endswith()

检查是否以指定字符结束，返回True/False

mystr.endswith(obj)

replace()

替换字符为指定字符

待添加

格式化

capitalize()

字符串第一个字符转为大写

待添加

title()

字符串中每个单词首字母转为大写

待添加

lower()

转换为小写

mystr.lower()

upper()

转换为大写

mystr.upper()

ljust()

返回一个原字符串左对齐，并使用空格填充至长度width的新字符串

mystr.ljust(width)

rjust()

返回一个原字符串右对齐，并使用空格填充至长度width的新字符串

mystr.rjust(width)

center()

返回一个原字符串居中，并使用空格填充至长度width的新字符串

mystr.center(width)

字符串——列表 转换

split()

字符串 转 列表

待添加

join()

列表 转 字符串

列表中，每个元素插入指定字符，组成新的字符串
list.join(str)

例：

*splitlines()

返回一个包含每行为元素的列表

mystr.splitlines()

空白字符

strip()

删除字符串两端的空白字符

mystr.strip()

lstrip()

删除字符串左边的空白字符

mystr.lstrip()

rstrip()

删除字符串末尾的空白字符

mystr.rstrip()

判断类型

isalpha()

判断字符串中所有字符是否为字母，返回True/False

mystr.isalpha()

isdigit()

判断字符串是否包含数字，返回True/False

mystr.isdigit()

isalnum()

判断字符串中所有字符是否为数字或字母，返回True/False

mystr.isalnum()

isspace()

判断字符串中包含空格，返回True/False

mystr.myspace()
