并发

当前任务数大于核数，一定有程序没有执行，叫做并发

并行

核数大于任务数，即任务可以永无休止的运行，且不需要轮转，叫做并行

*fork()

python的os模块封装fork()方法，使用fork()方法创建子进程
注意：fork函数在windows系统不可以使用

使用方法

分析：

  • 程序执行到os.fork()，操作系统创建一个新的进程（子进程），然后复制父进程的所有信息到子进程
  • 父进程和子进程都会从fork()函数中得到一个返回值，在子进程为0，在父进程中是子进程的ID号

getpid()、getppid()

getpid()获取当前pid，getppid()父进程pid

使用方法

进程最大值：65535

多进程修改全局变量

分析：

多进程中，每个进程的所有数据（包括全局变量）都各自拥有一份，互不影响
