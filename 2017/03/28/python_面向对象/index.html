深拷贝、浅拷贝（爬虫使用深拷贝）

###浅拷贝

拷贝引用，但不拷贝内容

###深拷贝

  1. 需要导入模块，import copy
  2. 拷贝：c = copy.deepcopy(a)
  3. 引用的内存地址不同

###区别

  • 把内容拷贝放入一片新内存空间（缺点：占内存）
  • 只拷贝引用（缺点：修改一个值，另外一个也改变，存在安全隐患）

###copy.copy()

  • 可变类型：创建新地址，类似深拷贝
  • 不可变类型：类似浅拷贝

私有化

  1. xx：公有变量
  2. _x：私有化属性或方法
    • from some import *不可导入
    • 类对象和子类可以访问
  3. __xx：避免与子类的属性命名冲突
    • 无法在外部直接访问
    • 子类不能继承、不能访问
    • 如果在子类中赋值，那么会在子类定义与父类同名的__xx
  4. xx：魔法属性或方法
    • 例如：init；
    • 注：不要自己创建此类属性和方法
  5. xx_：用于避免与python关键字冲突
