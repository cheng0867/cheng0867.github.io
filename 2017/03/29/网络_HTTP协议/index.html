HTTP协议

HTTP请求方式

  • GET：获取数据
  • POST：修改数据
  • PUT：保存数据
  • DELETE：删除数据
  • OPTION：询问服务器的某种支持特性
  • HEAD：返回报文头

状态码 status code

  • 2开头代表连接成功
  • 4开头代表客户端错误
  • 5开头代表服务端错误

URL（URI）、URN

全局资源定位标识

  • L:location 位置
  • N:name 名字

HTTP特性

无状态：并不知道每次连接的前后关系，即每次连接后就会断掉
