Linux系统操作指南

发表于 2017-04-07 | 分类于 Linux

linux系统操作

认识Linux

常用Linux版本

  • Ubuntu
  • Redhat
  • CentOS

应用领域

  • 个人桌面：如ubuntu
  • 服务器：如Redhat
  • 嵌入式：如机顶盒、数字电视、手持设备

用户目录

工作目录（家目录）

绝对路径

以/开头的路径为绝对路径

相对路径

.和..

.表示当前目录
..表示上一级目录，即父目录

访问

访问用户

三种访问方式限制访问权限

  • 所有者：只允许用户自己访问，所有者就是创建文件的用户
  • 用户组：允许一个预先指定的用户组中访问，用户都组成用户组
  • 其他用户：允许系统中的任何用户访问

访问权限

用户可以控制一个文件或目录的访问程度

  • 读权限（r）:对文件，有读取文件内容的权限；对目录，有浏览目录的权限
  • 写权限（w）：对文件，有新增、修改文件内容的权限；对目录，有删除、移动目录内文件的权限
  • 执行权限（x）：对文件，有执行文件的权限；对目录，有进入目录的权限

命令使用方法

Linux命令格式

  • command：命令名
  • options：选项，用对命令进行控制，也可省略
  • parameter1 … ：传给命令的参数，可有可无
  • []：代表可选

查看帮助文档

–help

Linux命令自带的帮助信息，如：ls –help

man

man是Linux提供的一个手册，包含了大部分的命令、函数使用说明，例：man ls；man 2 printf

man快捷键
|功能键|功能|
|-|-|
|空格键|显示手册页的下一屏|
|Enter键|一次滚动手册也的一行|
|b|回滚一屏|
|f|前滚一屏|
|q|退出man命令|
|h|列出所有功能键|
|/word|搜索word字符串|

自动补全

输出前几个字母时，按下tab键。系统会自动补全命令

历史命令

使用上、下键翻看之前的命令历史

文件管理

查看文件信息：ls

ls是list的缩写，列出目录的内容，类似DOS下的dir命令

ls常用参数
|参数|含义|
|-|-|
|-a|显示指定目录下所有子目录与文件，包括隐藏文件|
|-l|以列表方式显示文件的详细信息|
|-h|以人性化方式显示文件大小|

通配符：使用特殊字符引用多个文件名

  • *：文件代表文件名中所有字符
    • ls te*：查找以te开头的文件
    • ls *html：查找结尾为html的文件
  • ?：代表文件名中任意一个字符
    • ls ? .c：只找第一个字符任意，后缀为.c的文件
    • ls a.?：只找只有3个字符，前2字符为a.，最后一个字符任意的文件
  • []：“[”和“]”f将字符组括起来，表示可以匹配字符组中的任意一个。“-”用于表示字符范围
    • [abc]：匹配a、b、c中的任意一个
    • [a-f]：匹配从a到f范围内的任意一个字符
    • ls [a-f]*：找到从a到f范围内的任意一个字符开头的文件
    • ls a-f：查找文件名为a-f的文件，当“-”处于方括号之外失去通配符的作用
  • \：转义字符，“?”和“*”处于方括号内时失去通配符作用
    • ls \*a：查找文件名为*a的文件

输出重动向命令：>

将命令执行结果重定向到一个文件，即本应显示在终端上的内容保存到指定文件中
如：ls > test.txt（test.txt如果不存在，则创建，存在，则覆盖内容）

  • >：输出重定向会覆盖原来的内容
  • >>：输出重定向会追加到文件的尾部

分屏显示：more

查看内容时，文件内容过多，可使用more方式查看，每次只显示一页

  • 翻页：空格键
  • 退出：q键
  • 帮助：h键

管道：|

一个命令的输出可以通过管道作为另一个命令的输入
其中：左端写，右端取

清屏：clear

清除终端上的显示，快捷键：ctrl + “ l ”

切换工作目录：cd

cd后面根绝对路径，或相对路径。

  • cd：当前用户主目录（/home/用户目录）
  • cd ~：当前用户主目录
  • cd .：当前目录
  • cd ..：上级目录
  • cd -：上次所在目录
  • cd /：根目录

显示当前路径：pwd

显示当前工作目录，不需参数

创建目录：mkdir

创建一个新目录
注意：1.不能与已有目录名同名；2.必须对当前目录有写权限

删除目录：rmdir

删除一个目录，必须离开目录，并且目录为空

删除文件：rm

删除一个文件或目录
注：小心使用，删除后不可恢复

  • -i：询问确认的方式执行
  • -f：强制删除，忽略不存在的文件，无提示
  • -r：递归删除目录下内容，删除目录必须使用该参数

建立链接文件：ln

链接两种方式

  • 软链接：不占用磁盘空间，源文件删除则链接失败（类似快捷方式），修改软链接，不影响源文件
  • 硬链接：只能链接普通文件，不能链接目录，修改硬链接，改变源文件

使用格式

查看或合并文件内容：cat

  • ：先清空，后添加

  • ：直接添加

文本搜索：grep

一般格式

在输入字符串参数时，最好使用引号括起来，例如：grep “hello world” test.txt

  • -V：显示不包括匹配文本的所有行（取反）
  • -n：显示匹配行及行号
  • -i：忽略大小写

grep搜索内容可以使用正则表达式
grep常用正则表达式

  • ^a：行首，搜索以a开头的行。如：grep -n “^a” 1.txt
  • ke&：行尾，搜索以ke结束的行。如：grep -n “ke$” 1.txt
  • [Ss]igna[Ll]：匹配[]中一系列字符中的一个。如：搜索匹配单词signal，signaL，Signal的行；grep -n “[Ss]igna[Ll]” 1.txt
  • .（点）：匹配一个非换行符的字符。如：匹配e和e之间有任意一个字符，可以匹配eee，eae，eve，但不可匹配ee，eaae；grep -n “e.e” 1.txt

查找文件：find

在特定的目录下搜索符合条件的文件

常用方法

命令 含义
find ./ -name test.sh 查找当前目录下所有名test.sh的文件
find ./ -name ‘*.sh’ 查找当前目录下所有名后缀为.sh的文件
find ./ -name “[A-Z]*” 查找当前目录下所有名以大写字母开头的文件
find /tmp -size 2M 查找在/tmp目录下等于2M的文件
find /tmp -size +2M 查找在/tmp目录下大于2M的文件
find /tmp -size -2M 查找在/tmp目录下小于2M的文件
find ./ -size +4k -size -5M 查找当前目录下大于4k，小于5M的文件
find ./ -perm 0777 查找当前目录下权限为777的文件或目录

拷贝文件：cp

将给出的文件或目录复制到另一个文件或目录中，相当于DOS下的copy命令

常用选项

  • a：保留链接、文件属性，并递归地复制目录，即保持文件原有属性
  • f：如已存在文件不提示
  • i：询问确认复制
  • r：如给出的源文件是目录文件，则cp将递归复制该目录下的所有子目录和文件
  • v：显示拷贝进度

移动文件：mv

移动文件或目录，也可以给文件或目录重命名

常用选项

  • -f：强制执行，如有覆盖不提示
  • -i：询问确认，如有覆盖则提示
  • -v：显示进度

归档管理：tar（压缩）

使用格式

注：tar命令很特殊，其参数前面可以使用“-”，也可以不使用

常用参数

  • c：生成档案文件，创建打包文件
  • v：显示进度
  • f：指定档案文件名称，f后面一定是.tar文件，所以必须放选项最后
  • t：列出档案中包含的文件
  • x：解开档案文件

注：除了f需要放在参数的最后，其它参数顺序任意

文件压缩解压：gzip

结合tar与gzip使用实现文件打包、压缩。
tar：只负责打包文件，但不压缩
gzip：压缩tar打包后的文件，其扩展名为：xxx.tar.gz

gzip使用格式

常用选项

  • -d：解压
  • -r：压缩所有子目录

虽然tar并没有压缩，只有打包功能，但添加-z选项可以调用gzip压缩功能

-z：指定压缩包的格式为：file.tar.gz

解压使用格式

文件压缩解压：bzip2

与gzip类似，不同之处，将-z改为-j

文件压缩解压：zip、unzip

通过zip压缩文件的目标文件不需要指定扩展名，默认扩展名为zip

使用格式

  • 解压文件：unzip -d 解压后目录文件 压缩文件

查看命令位置：which

使用which查看命令位置，如果找到就显示这个命令的路径

用户、权限管理

用户管理包括用户与组账号的管理

  • 标准用户：默认的用户，权限有限
  • 管理员：root账户，不受权限限制

查看当前用户：whoami

用于查看当系统当前账号的用户名，可通过cat /etc/passwd查看系统用户信息

查看登录用户：who

用户查看当前所有登录系统的用户信息

常用选项

  • -m或am i：只显示运行who命令的用户名、登录终端和登录时间
  • -q或–count：只显示用户的登录账号和登录用户的数量
  • -u或–heading：显示列标题

退出登录账户：exit

不同情况下，退出结果不同

  • 图形界面：退出当前终端
  • ssh远程登录：退出登录账户
  • 切换后的登录用户：退出则返回上一个登录账号

添加用户账号：useradd

添加用户账号可使用adduser或useradd命令，因为adduser命令是指向useradd命令的一个链接，所以两个命令的使用格式完全一样

使用格式

常用参数

  • -d：指定用户登录时的主目录，如不指定，默认在/home目录下创建与用户名同名目录作为主目录
  • -m：自动建立目录
  • -g：指定组名称

设置用户密码：passwd

超级用户可以使用passwd命令为普通用户设置或修改用户口令

删除用户：userdel

  • userdel abc(用户名)：删除abc用户，但不会自动删除用户的主目录
  • userdel -r abc(用户名)：删除用户，同时删除用户的主目录

切换用户：su

su后面可以加“-”，可直接切换到对应用户的主目录（家目录）

  • su：切换到root用户
  • su root：切换到root用户
  • su -：切换到root用户，同时切换目录到/root
  • su - root：切换到root用户，同时切换目录到/root
  • su 普通用户：切换到普通用户
  • su - 普通用户：切换到普通用户，同时切换普通用户所在的目录

注：需要sudo配合使用

ubuntu下切换到root的简单命令：sudo -s

修改文件权限：chmod

字母法：chmod u/g/o/a +/-/= rwx 文件

[u/g/o/a] 含义
u user表示该文件的所有者
g group表示与该文件的所有者属于同一组(group)者，即用户组
o other表示其他以外的人
a all表示这三者皆是
[+-=] 含义
+ 增加权限
- 撤销权限
= 设定权限
rwx 含义
r read表示可读，如果没有r，就不能通过ls查看这个目录的内容
w write表示可写，如果没有w，就不能在目录下创建新的文件
x excute表示可执行，如果没有x，就不能通过cd进入该目录

数字法：“rwx”这些权限也可以用数字来代替

字母 说明
r 读取权限，数字代号为”4”
w 写入权限，数字代号为”2”
x 执行权限，数字代号为”1”
- 不具任何权限，数字代号为”0”

如：chmod u=rwx,g=rx,o=r filename 等同于：chmod u=7,g=5,o=4 filename

chmod 751 file：

  • 文件所有者：读、写、执行权限
  • 同组用户：读、执行权限
  • 其他用户：执行权限

注：递归添加权限，使用“-R”

系统管理

查看当前日历：cal

cal命令用于查看当前日历，选项-y显示整年日历

显示或设置时间：date

设置时间格式（需要管理员权限），如：date 010203042016.55

  • CC：年的前两位
  • yy：年的后两位
  • MM：月
  • mm：分钟
  • dd：天
  • hh：小时
  • ss：秒

显示时间格式（date’+%y, %m, %d, %H, %M, %S’）
format格式：

  • %Y, %y：年
  • %m：月
  • %d：日
  • %H：时
  • %M：分
  • %S：秒

查看进程信息：ps

进程是具有一定独立功能的程序，是操作系统动态执行的基本单元

常用选项（可不加“-”）

  • -a：显示终端上的所有进程，包括其他用户的进程
  • -u：显示进程的详细状态
  • -x：显示没有控制终端的进程
  • -w：显示加宽，以便显示更多的信息
  • -r：只显示正在运行的进程

动态显示进程：top

在top命令运行后，在指定的时间间隔更新显示信息。可使用top + -d来指定更新间隔

常用按键

  • M：根据内存使用量来排序
  • P：根据CUP占有率来排序
  • T：根据进程运行时间的长短来排序
  • U：根据输入的用户名筛选进程
  • K：根据输入的PID来杀死进程
  • q：退出
  • h：帮助

终止进程：kill

kill命令指定进程号的进程，需要配合ps使用
使用格式

注：signal代表信号值，从0到15，其中9为绝对终止，即使用-9强制杀死进程

关机重启：reboot、shutdown、init

  • reboot：重启操作系统
  • shutdown -r now：重启操作系统，shutdown会给别的用户提示
  • shutdown -h now：立刻关机，其中now相当于时间为0的状态
  • shutdown -h 20:25：系统在今天的20:25会关机
  • shutdown -h +10：系统再过10分钟自动关机
  • init 0：关机
  • init 6：重启

检测磁盘空间：df

用于检测文件系统的磁盘空间占用和空余情况，可以显示所有文件系统对节点和磁盘块的使用情况

常用选项

  • -a：显示所有文件系统的磁盘使用情况
  • -m：以1024字节为单位显示
  • -t：显示各指定文件系统的磁盘空间使用情况
  • -T：显示文件系统

检测目录所占磁盘空间：du

用于统计目录或文件所占磁盘空间的大小，与df类似，但du侧重磁盘使用情况

使用格式

常用选项

  • -a：递归显示指定目录中各文件和子目录中文件占用的数据块
  • -s：显示指定文件或目录占用的数据块
  • -b：以字节为单位显示磁盘占用情况
  • -l：计算所有文件大小，对硬链接文件计算多次

查看或匹配网卡信息：ifconfig

默认该指令显示所有网卡信息
