Emmet快捷操作

发表于 2017-04-09 | 分类于 工具

Emmet插件可自动补全代码

语法

子节点：>

nav>ul>li

兄弟节点：+

div+p+bq

提升：^

div+div>p>span+em^bq 其中^左边内容在上，右边内容在下

div+div>p>span+em^^bq 使用^^跳出当前父级节点

分组：()

div>(header>ul>li*2>a)+footer>p

(div>dl>(dt+dd)*3)+footer>p

倍数：*

ul>li*5

项编号：$

ul>li.item$*5

h$[title=item$]{Header $}*3

ul>li.item$$$*5

ul>li.item$@-*5 其中@-转为倒序

ul>li.item$@3*5 从3起始，个数为5

ID和Class属性

#header

.title

form#search.wide

p.class1.class2.class3

自定属性

p[title="hello world"]

td[rowspan=2 colspan=3 title]

[a="value1" b="value2"]

文本：{}

a{click me}

p>{click}+a{here}+{to continue}

隐式标签名

.class1默认为div标签

em>.itemem中默认为span标签

ul>.item*3ul中默认为li标签

table>.row>.coltable标签中默认为tr标签，其次默认为td标签

HTML

所有未知缩写都会变成标签，如：foo ——> <foo></foo>

! 构建html 5基础结构，别名：html:5
