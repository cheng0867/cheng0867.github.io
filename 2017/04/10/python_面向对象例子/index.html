烤地瓜

属性

  • cookedLevel：值为数字；0~3表示生的（初始值），3~5表示半生不熟，5~8表示熟了，超过8表示烤焦了
  • cookedString：值为字符串；描述生熟程度
  • condiments：额外添加的调料

方法

  • cook()：烤方法，把地瓜烤一段时间
  • addCondiments()：给地瓜添加调料
  • __init__()：设置默认的属性
  • __str__()：让print的结果看起来更好一些
