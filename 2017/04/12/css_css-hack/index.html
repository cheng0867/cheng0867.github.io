透明度兼容

在IE下img标签默认有1像素边框

h标签继承body中的font-size设置

a标签中使用img后的高度多了几个像素

解决办法

  1. 消除掉匿名盒子的高度，也就是给a设置line-height:0或font-size:0;
  2. 给两者vertical-align:top，让其top对齐，而不是baseline对齐
  3. 给img以display:block，让它和匿名行级盒子不在一个布局上下文中，也就不存在行级盒的对齐问题
  4. 因为img是行内元素，默认display: inline; 它与文本的默认行为类似，下边缘是与基线对齐，而不是紧贴容器下边缘。将displayp设置为block即可消除上面说的几个像素的差别
