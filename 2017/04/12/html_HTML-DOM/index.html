HTML-DOM

发表于 2017-04-12 | 分类于 html

HTML DOM是关于如何获取、修改、添加、删除 HTML元素的标准

  • HTML的标准对象模型
  • HTML的标准编程接口
  • W3C标准

HTML DOM 节点

DOM 节点

根据W3C标准，HTML文档中的所有内容都是节点

  • 文档节点：整个文档
  • 元素节点：每个HTML元素
  • 文本节点：HTML元素内的文本
  • 属性节点：每个HTML属性
  • 注释节点：注释

HTML DOM 节点树

HTML DOM 将 HTML 文档视作树结构。这种结构被称为节点树

HTML DOM 树的所有节点均可通过 JavaScript 进行访问。所有HTML元素（节点）均可被创建、修改、或删除

节点关系：父、子和同胞

节点树中的节点彼此拥有层级关系

  • 在节点树中，顶端节点被称为根节点（root）
  • 每个节点都有父节点，除了根节点
  • 一个节点可拥有任意数量的子节点
  • 同胞是拥有相同父节点的节点

通过innerHTML属性来访问文本节点的值

HTML DOM 方法

方法就是可以在节点（HTML元素）上执行的动作

编程接口

调用HTML元素对象的属性和方法，就是调用该元素对象的编程接口

  • 可通过JavaScript（或其它语言）对HTML DOM进行访问
  • 所有HTML元素被定义为对象， 而编程接口则是对象方法和对象属性
  • 方法是能够执行的动作（如对元素的增、删、改、查）
  • 属性是能够获取或设置的值（如节点名称、内容、属性）

getElementById()方法

getElementById()方法返回带有指定ID的元素

HTML DOM 对象 - 方法和属性

常用HTML DOM方法：

  • getElementById(id) - 获取带有指定ID的节点（元素）
  • appendChild(node) - 插入新的子节点（元素）
  • removeChild(node) - 删除子节点（元素）

常用HTML DOM属性：

  • innerHTML - 节点（元素）的文本值
  • parentNode - 节点（元素）的父节点
  • childNodes - 节点（元素）的子节点
  • attributes - 节点（元素）的属性节点

常用DOM对象方法

方法 描述
getElementById() 返回带有指定ID的元素
getElementsByTagName() 返回包含带有指定标签名称的所有元素的节点列表
getElementsByClassName() 返回包含带有指定类名的所有元素的节点列表
appendChild() 把新的子节点添加到指定节点
removeChild() 删除子节点
replaceChild() 替换子节点
insertBefore() 在指定的子节点前面插入新的子节点
createAttribute() 创建属性节点
createElement() 创建元素节点
createTextNode() 创建文本节点
getAttribute() 返回指定的属性值
setAttribute() 把指定属性设置或修改为指定的值

HTML DOM 属性

属性是节点（HTML元素）的值，可获取和设置

innerHTML属性

该属性是获取或替换元素内容最简单的方法

nodeName属性

nodeName属性规定节点的名称

  • nodeName是只读的
  • 元素节点的nodeName与标签名相同
  • 属性节点的nodeName与属性名相同
  • 文本节点的nodeName始终是#text
  • 文档节点的nodeName始终是#document

nodeValue属性

nodeValue属性规定节点的值

  • 元素节点的nodeValue是undefined或null
  • 文本节点的nodeValue是文本本身
  • 属性节点的nodeValue是属性值

nodeType 属性

nodeType 属性返回节点的类型（nodeType是只读的）

比较重要的节点类型有：

  • 元素：1
  • 属性：2
  • 文本：3
  • 注释：8
  • 文档：9

访问 HTML 元素（节点）

访问HTML元素相当于访问节点

访问方式：

  • 通过使用getElementById()方法
  • 通过使用getElementByTagName()方法
  • 通过使用getElementByClassName()方法
