gulp自动化构建

发表于 2017-04-15 | 分类于 工具

前端自动化工具

前提

安装npm

需要的gulp插件

  • gulp-jade：编译jade文件为html文件
  • gulp-stylus：编译stylus文件为css文件
  • browser-sync：浏览器同步测试工具

新建一个项目

目录结构
project
|——web //存放预编译代码
|——app //存放编译后代码

在项目下安装gulp

创建package.json文件（用来保存项目相关信息）

$ npm init

全局安装

$ npm install gulp -g

局部安装

$ npm install gulp --save-dev

安装所需插件

$ npm install gulp-stylus gulp-jade browser-sync --save-dev

如果有错误，可尝试单独安装

实例化插件

在项目下创建gulpfile.js(必须是这个名字)

创建gulp任务
