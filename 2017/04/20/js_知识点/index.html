JavaScript知识点

发表于 2017-04-20 | 分类于 JavaScript

综合梳理JavaScript常见知识点

数据类型

number

JavaScript不区分整数和浮点数，统一用number表示

字符串

""和''本身只是一种表示方式，不是字符串的一部分

布尔值

==和===

==：自动转换数据类型，再比较，容易产生错误结果
===：不会自动转换数据类型，如果数据类型不一致，返回false，如果一致，再比较

注：不要使用==比较，始终坚持使用===比较

NaN

NaN是number中的一个特殊，与其它值都不相等，包括它自己

唯一能判断NaN的方法是通过isNaN()函数

因为计算机对浮点数精确计算存在误差。所以要比较两个浮点数是否相等，只能计算它们之差的绝对值，看是否小于某个阈值

null和undefined

null：表示一个空的值
undefined：表示值未定义
0：表示一个数值
""：表示长度为0的字符串

null与undefined两者区分不大，大多数情况使用null，undefined只在判断函数参数是否传递的情况下使用

数组(类似python中的list)

js数组可以包含任意数据类型

两种声明方式

  • 直接声明

  • 使用Array()函数实现

对象(类似python中的dict)

js对象是一组由键-值组成的无序集合

变量

变量保存任意数据类型

命名规则

  • 大小写英文
  • 数字
  • $或_组合
  • 不能用数字开头

示例

动态语言 与 静态语言 区别

  • 动态语言：同s一变量，可以反复赋值，而且可以是不同类型的变量
  • 静态语言：在定义变量时必须指定变量类型，如果赋值时类型不匹配，就会报错

strict模式

  • 全局变量：如果一个变量没有通过var声明就被使用，那么该变量就自动被声明为全局变量
  • 局部变量：使用var声明的变量被限制在变量被声明的函数体内，同名变量在不同的函数体内互不冲突

在strict模式下，强制通过var声明变量，未使用var声明变量就使用的，将导致运行错误

注：为避免这一缺陷，所有的js代码都应该使用strict模式

字符串

js的字符串就是用''或""括起来的字符表示

如果字符串内包含'或"，可以使用转义字符\来标识

常见转义字符

  • \n：表示换行
  • \t：表示制表符
  • \\：表示转义\

多行字符串

由于多行字符串使用\n比较麻烦

ES6标准新增一种多行字符串的表示方法，用反引号 ` … ` 表示

模版字符串

要把多个字符串连接起来，可以用+号连接

ES6标准新增了一种模版字符串，表示方法和普通的多行字符串一样，但是会自动替换字符串中的变量

数组

js的Array可以包含任意数据类型，并通过索引来访问每个元素

获取数组长度

直接访问length属性

注：直接给Array的length赋值会导致Array大小的变化（不推荐使用）

数组通过索引修改值

注：通过索引赋值超出范围，同样会引起Array大小的变化（不推荐使用）

搜索位置 indexOf()

搜索一个指定的元素的位置，找到返回下标，未找到返回-1

截取 slice()

类似字符串的substring()，截取Array的部分元素，返回一个新的Array

如果不给slice()传递任何参数，它就会从头到尾截取所有元素。利用这一点，我们可以很容易地复制一个Array

头部添加和删除 unshift() and shift()

unshift()向Array的头部添加若干元素，shift()把Array的第一个元素删除

排序 sort()

sort()可以对当前Array进行排序，它会直接修改当前Array的元素位置，直接调用时，按照默认顺序排序

反转 reverse()

reverse()把整个Array反转

修改 splice()

splice()是修改Array的“万能方法”，从指定的索引开始删除若干元素，然后再从该位置添加若干元素

连接 concat()

把当前Array和另一个Array连接起来，并返回一个新的Array

注：concat()方法并没有修改当前Array，而是返回一个新的Array
并且：concat方法可以接收任意个元素和Array，并且自动把Array拆开，然后全部添加到新的Array里

合并字符串 join()

把当前Array的每个元素都用指定的字符串连接起来，然后返回连接后的字符串

注：如果Array的元素不是字符串，将自动转换为字符串后再连接

多维数组

如果数组的某个元素又是一个Array，则可以形成多维数组

对象

JS的对象是一种无序的集合数据类型，它由若干键值对组成

访问属性是通过.操作符完成的，但这要求属性名必须有一个有效的变量名。如果属性名包含特殊字符，就必须用''括起来
