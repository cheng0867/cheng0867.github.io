HTTP协议主要请求方式：GET、POST

HttpRequest对象

构造一个对象， 将请求报文赋给该对象

  • path：请求的完整路径
  • method：GET通过链接方式,POST通过表单方式
    GET将参数放在url中，
    POST将参数放在请求报文，
  • encoding：编码，默认None就是UTF-8编码
  • Cookie：缓存，用于保存用户状态信息，存储在浏览器
  • Session：缓存，用于保存用户敏感信息，存储在服务器

  • content-type：返回数据的MIME类型

方法：

  • init：
  • set_cookie：设置Cookie信息
  • write：向相应体中写数据

  • cookie：以键值对格式存储在浏览器中的一段纯文本信息，作用是实现用户的跟踪

    • max_age：指定秒后过期

删除cookie：delete_cookie
查询cookie：request.COOKIE[‘x’]

  1. 在服务器中写入状态（Session）

    对于敏感信息，需要使用Session存入服务器中
    Session依赖Cookie
