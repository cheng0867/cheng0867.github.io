模版包含两部分：
静态部分：html、css、js
动态部分：模版语言

变量

在双花括号内，获取变量值并输出

解析过程（顺序），例如book.title

1.先解析book.title，如果没有再book[‘title’]
2.先属性后方法，先找title属性，没有再找title()方法
3.如果book.0则解析为book[0]

过滤器

使用管道符号|来应用过滤器，用于计算、转换操作，可以在变量和标签中使用
如果使用参数，使用冒号：传递参数

  • 长度length：表示字符串包含字符的个数，或列表、元组、字典的元素个数
  • 默认值default：如果变量不存在则返回默认值

日期date：用于对日期格式化

时间由settings中的时区决定

  • Y：年，4位
  • y：年，2位
  • m：月，格式为01,02,12
  • j：日，格式为1,2
  • H：小时，24制
  • h：小时，12制
  • i：分，0-59
  • s：秒，0-59

自定义过滤器

  1. 在应用中创建目录templatetags
  2. 使用init.py使目录转为模块package，内容为空

  3. 在templatetags目录中创建filters.py（文件名随意）

  4. 在html页面中引用自定义过滤器
