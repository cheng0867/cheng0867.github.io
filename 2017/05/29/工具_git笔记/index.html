创建仓库

Create a new repository

  1. Repository name：django1
  2. Public：checked
  3. Initiaalize this repository with a README：checked

create new file

  1. 文件名：.gitignore（固定名字）
  2. Edit new file：*.pyc
  3. 点击Preview查看结果
  4. 点击Commit new file提交

添加SSH账户

如果一台计算机需要与github上的仓库交互，需要把这台计算机的ssh公匙添加到github账户上

  1. 展开账户头像下拉菜单，点击settings
  2. 选择SSH and GPG keys
  3. 点击New SSH key

  4. 在ubuntu的命令行中，修改计算机的git配置

  5. 生成git密匙：公匙为id_rsa.pub / 密匙为id_rsa

  6. 查看公匙并复制内容

克隆项目

  1. 进入仓库
  2. 点击Clone or download，再点击use ssh复制git地址
  3. 在终端中复制仓库中的内容

创建项目分支

  • 开发期间每个员工代码互不干扰，所以每人使用一条分支
  • 项目开发中公用分支包括master、dev
    • 分支master用于发布，默认分支，当需要发布时将dev分支合并
    • 分支dev开发阶段的代码合并，每个阶段的工作完成之后需要进行一次合并，控制项目的进度
    • 成员分支用于每个项目成员的代码开发，实现不交叉
  • 创建分支

共有分支需要跟踪，个人分支无需跟踪
