Git笔记2

发表于 2017-05-29 | 分类于 工具

Git是目前世界上最先进的分布式版本控制系统（没有之一）。

安装Git

安装git后需要设置用户名和邮箱

创建版本库

  1. 选择一个合适的地方，创建一个空目录

  2. 使用git init将目录编程Git仓库

此时该目录下多了一个.git目录，该目录是Git来跟踪管理版本库的，不要轻易修改该目录内容

  1. 在learngit目录下创建readme.txt文件

  2. 添加readme.txt到Git仓库（把一个文件放到Git仓库的方法）

执行该步操作没有任何显示，就是正确的

  1. 使用git commit把文件提交到仓库

其中：-m后面输入的是本次提交说明，可输入任意内容
为什么Git添加文件需要add、commit两步呢？因为commit可以一次提交多个文件

回滚

  1. 修改readme.txt文件

  2. 使用git status命令查看结果

  3. 提交修改git add

  4. 查看结果git status

已知要提交的修改包括readme.txt，可以执行提交操作了

  1. 提交git commit

  2. 查看结果git status
