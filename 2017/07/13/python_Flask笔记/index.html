问题：Flask和Django区别？
答案：
Django提供完整解决方案，MVT模式，ORM，后台管理
Flask轻巧、简洁，可扩展性高

问题：访问“百度”时的具体流程？
答案：

  1. 输入网址，按下回车后
  2. 客户端浏览器将域名转发给DNS进行域名解析
  3. DNS返回具体的IP地址
  4. 客户端向该地址的服务器发送tcp三次握手
  5. 客户端发送http请求
  6. 静态资源：服务器直接返回，图片、JS、CSS、状态码
  7. 动态资源：服务器查询数据库，返回结果，在通过逻辑处理后，返回客户端
